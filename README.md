**CodeFlow** es una plataforma colaborativa de gestión de proyectos diseñada específicamente para la comunidad estudiantil de la **Universidad Nacional Autónoma de Honduras (UNAH)**. Basada en metodologías ágiles, permite a los equipos de software planificar, organizar y rastrear su progreso de manera profesional.

---

##  Características Principales

-  **Tablero Kanban Interactivo** con funcionalidad *Drag & Drop*
-  **Product Backlog** global para la gestión de Historias de Usuario
-  **Notificaciones en Tiempo Real** mediante WebSockets
-  **Analítica y Métricas** — Burndown Chart y rendimiento por usuario
-  **Historial de Actividad** con trazabilidad completa del tablero
-  **Control de Roles** — ADMIN, COLABORADOR y LECTOR
-  **Exportación de Reportes** en formato PDF y Excel
-  **Autenticación segura** mediante JWT y Spring Security

---

##  Stack Tecnológico

**Backend** — Java 17, Spring Boot 4.0.4, Spring Security, MySQL 8.0, STOMP/SockJS, Apache POI, iText

**Frontend** — React 19, Vite, Tailwind CSS 4.2, dnd-kit, Recharts, Axios

**Infraestructura** — Docker Desktop
